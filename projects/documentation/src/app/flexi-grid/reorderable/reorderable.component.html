Kodların açıklaması

  • [reorderable] : Sıralama özelliğini aktif etmek istiyorsak true, pasife çekmek istiyorsak false vermeliyiz
  • (onReorder) Sırası değişen satırın önceki ve sonraki değerini bize verir
  • [reOrderWidth] : Kolon genişliğini bunun ile ayarlayabilyoruz.
  • [reOrderTextAlign] : Kolon içindeki iconun sağ sol ya da ortada duracağını bununla belirliyoruz
Attribute Alabileceği Değer Türleri Default Değer
reorderable true | false false
reOrderWidth string 50px
reOrderTextAlign left | center | right center

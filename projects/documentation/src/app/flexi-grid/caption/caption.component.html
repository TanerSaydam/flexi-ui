Kodların açıklaması

  • [showCaption] : Caption bölümünü göstermek için bu attribute true değer vermelisiniz
  • captionTitle (optional): Caption'a title vermek için bu attribute değer atayabilirsiniz
  • [showColumnVisibility] (optional) : Caption bölümü gözüküyorken columnların visibilitysini yöneten bu dropdownu gösterip gizleyebilirsiniz
  • columnVisibilityBtnClass : Column Visibility butonu için özel class vermek isterseniz bu attribute kullanabilirsiniz
  • [showRefreshBtn] (optional) : Caption bölümü gözüküyorken datanın refresh olmasını sağlayan bu butonu gösterip gizleyebilirsiniz. Bu butona tıkladığınızda tüm parametreleri sıfırlar. Order, filter vs gibi.
  • refreshDataBtnClass : Refresh Data butonu için özel class vermek isterseniz bu attribute kullanabilirsiniz
  • ng-template #captionTemplate : Bu elementi oluşturup içerisine eklediğiniz her şey caption sağ taraftaki button kısmında gözükür
  • [captionTemplate] : Eklediğiniz buttonların gözükmesi için bu elementin referansını attirbute vermeniz gerekmektedir

    Not: ng-template #captionTemplate ile element oluşturmanız içerisine ekleyeceklerinizi captionda göstermek için yeterli değil. Mutlaka burada hashtag ile elde ettiğiniz referansı [captionTemplate] attribute değer olarak vermelisiniz!

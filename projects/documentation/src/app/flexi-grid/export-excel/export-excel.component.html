Kodların açıklaması

  • [showExportExcel] : Export Excel butonunu görmek için bu attribute true değer vermelisiniz
  • exportExcelFileName : Export edeceğiniz excele özel bir isim vermek için bu attribute değer atayabilirsiniz. Boş geçerseniz export-excel değerini kullanır
  • exportExcelButtonClick : Normalde export excel butonuna bastığınızda data griddeki mevcut veriyi indirir. Ama siz başka veriyi excele almak isterseniz bu attribute bir metot atayarak orada önce veriyi yakalarsınız, arkasına Flexi Service içerisindeki exportExcel metoduna yeni datanızı yollarak excel export yapabilirsiniz
Attribute Alabileceği Değer Türleri Default Değer
showExportExcel true | false false
exportExcelFileName text excel-export
exportExcelButtonClick () => void | null null

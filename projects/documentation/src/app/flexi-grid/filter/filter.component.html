Kodların açıklaması

  • [filterable] : Filter yapısını aktif etmek istiyorsak true, pasife çekmek istiyorsak false vermeliyiz
  • [filterable] (column) : Column bazlı filter yapısını kapatıp açabiliriz
  • [filterType] (column) : Column bazlı filter tipini değiştirebiliriz. Default tip text ama tarih, tarih ve saat ve number değerler için özel filter tipi verebiliriz. Verebileceğimiz tipler: text | date | date-time | number.

    Not: Filter tipini değiştirdiğimizde filitreleme yaptığımız seçenekte değişiyor. Text filtrelerken benziyor | benzemiyor iken, number filtresinde eşit | büyük | küçük gibi seçenekler geliyor. Date seçeneğinde filtre seçeneği sadece bir tane, o da eşit ise.

  • format (column) : FilterType date ya da number verdiyseniz columnu formatlayabilirsiniz. date için (d = gün || M = ay || y = yıl) formatı ile bir combinasyon yazabilirsiniz. Örnek: format="dd.MM.yyyy". Eğer number verdiyseniz n ya da c formatı seçenbilirsiniz. n = number, c = currency. n seçtiyseniz fraction (küsürat) sayısını belirleyebilirsiniz (default 2). c seçtiyseniz symbol, sembolün ön kısımdaki arkada mı görüneceği ve fraction belirleyebiliyorsunuz.

Kodların açıklaması

  • [showIndex] : Index sırasını görmek isterseniz bu attribute true değeri vermelisiniz
  • indexWidth : Index uzunluğu default 70px. Eğer değiştirmek isterseniz bu attribute kullanabilirsiniz

    Not: showIndex true değilse bu attribute verdiğiniz değer çalışmaz

  • indexTextAlign : Index'in text align olarak nerede duracağını belirler

    Not: showIndex true değilse bu attribute verdiğiniz değer çalışmaz

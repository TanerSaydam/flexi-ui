Kodların açıklaması

  • [pagination] : Paginationu aktif etmek istiyorsak true | pasife çekmek istiyorsak false vermeliyiz
  • [pageSize] (optional) : Page size'ıbir değişkene bağlayıp kendiniz başlangıç değeri verebilirsiniz. Değer olarak number bir değer vermeniz yeterli
  • [pageSizeList] (optional) : Page size listesini özelleştirmek isterseniz bu attribute değer olarak number bir liste vermelisiniz
Attribute Alabileceği Değer Türleri Default Değer
pagination true | false false
pageSize number 10
pageSizeList number[] [5, 10, 20, 30, 50, 100, 500, 1000]

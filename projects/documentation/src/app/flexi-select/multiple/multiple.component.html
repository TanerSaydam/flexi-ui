Kodların açıklaması

  • [multiple] : Çoklu seçimi aktif etmek için true değeri vermelisiniz
  • [closeAfterSelect] : Seçim sonunda select açılsın isterseniz bu attribute'e false değer gönderin. Default true olduğu için her seçim sonrası selecti kapatır
  • height : Selectin yüksekliğini ayarlayabilirsiniz. Min yükseklik 36px'dir.

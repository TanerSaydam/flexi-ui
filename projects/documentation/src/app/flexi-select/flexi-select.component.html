Angular uygulamalarınızda kullanabileceğiniz, şık ve kullanışlı bir Select olan Flexi select ile tanışın. Bu component ile listelerinizi select içerisinde gösterirken search yapabileceksiniz, lazy loading yapısı sayesinde listelerinizi göstermede performans elde edeceksiniz.

Flexi Select, açık kaynak kodludur. Kaynak kodlarına buraya tıklayarak ulaşabilirsiniz.

Projeye katkıda bulunmak ve bana destek vermek için GitHub reposuna bir yıldız (⭐) atmayı unutmayın. GitHub reposuna ulaşmak için tıklayın.

Flexi Select'i kullanmak istediğiniz standalone componente import etmelisiniz

HTML de component elementini çağırıp içine datayı vermelisiniz, arkasına selecte gözükmesini istediğiniz alanı label'a, size vermesini istediğiniz alanı value attribute'üne atamalısınız

component.ts

Ya da

Kodların açıklaması

  • flexi-select : Flexi select elementi. Bununla selecti sayfamıza ekliyoruz
  • [data] : Data attributene listemizi veriyoruz.

    Not: Projede değişkenler signal yapısında tutuldu ama zorunlu değildir. İsterseniz siz normal değişken kullanabilirsiniz

  • label : Select'de gözükmesini istediğimiz property ismini buraya veriyoruz
  • value : Her bir kaydın unique değerini, yani bizim için id niteliği taşıyan property ismini buraya veriyoruz.
  • [(ngModel)] : İstersek selecte bir değişkeni two-data-binding ile bağlayabiliyoruz
  • selected($event) : Seçtiğimiz değeri seçimin hemen sonrası bu event ile yakalayabiliyoruz
Attribute Alabileceği Değer Türleri
data object[] | number[] | string[] | boolean[] vs.
label text
value text

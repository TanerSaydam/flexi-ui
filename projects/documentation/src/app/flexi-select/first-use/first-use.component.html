Kodların açıklaması

  • flexi-select : Flexi select elementi. Bununla selecti sayfamıza ekliyoruz
  • [data] : Data attributene listemizi veriyoruz.

    Not: Projede değişkenler signal yapısında tutuldu ama zorunlu değildir. İsterseniz siz normal değişken kullanabilirsiniz

  • label : Select'de gözükmesini istediğimiz property ismini buraya veriyoruz
  • value : Her bir kaydın unique değerini, yani bizim için id niteliği taşıyan property ismini buraya veriyoruz.
  • [(ngModel)] : İstersek selecte bir değişkeni two-data-binding ile bağlayabiliyoruz
  • selected($event) : Seçtiğimiz değeri seçimin hemen sonrası bu event ile yakalayabiliyoruz

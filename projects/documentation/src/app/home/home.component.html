Flexi UI nedir?

Flexi UI bir Angular UI kütüphanesidir. İçerisinde ücretsiz şekilde kullanabileceğiniz UI componentleri yer almaktadır.

Angular kullanarak tasarlamak istediğimiz güçlü frontendler için en çok ihtiyaç duyduğumuz tasarımları bu UI kütüphanesi altında sizlerle paylaşmaya çalışacağım.

Tasarımların tamamı benim olup, tüm kodlar açık kaynaklı paylaşımdadır.

İsterseniz sizlerde kodlara destek olabilir, öneri ve destekleriniz ile bu UI kütüphanesinin büyümesine fayda sağlayabilirsiniz.

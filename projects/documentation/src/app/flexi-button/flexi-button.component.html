Angular uygulamalarınızda kullanabileceğiniz, şık ve kullanışlı buttonlar. Data Grid'e uygun olsun diye yazdım ama isterseniz proje içerisinde de kullanabilirsiniz.

Flexi Button'ı kullanmak/güncellemek için Angular projenizde aşağıdaki npm komutunu çalıştırmalısınız

npm install flexi-button

Not: Iconlar google iconu. Aşağıdaki linkten beğendiğiniz iconun adını yazmanız yeterli. Örneğin artı işareti için add gibi.

Google Iconlarını görmek için tıklayın
Attribute Alabileceği Değer Türleri Default Değer
btnIcon Tüm google iconlarını kullanabilirsiniz
btnSize small | default | large default
btnText text
